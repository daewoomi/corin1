GitHub GitHub is used both for issue tracking and test submissions; we provide a limited introduction to both git and GitHub. Pull Requests are automatically labeled based on the directory the files they change are in;

there are also comments added automatically to notify a number of people: this list of people comes from META.yml files in those same directories and their parents (i.e., they work recursively: a/META.yml will get notified for a/foo.html and a/b/bar.html). If you want to be notified about changes to tests in a directory, feel free to add yourself to the META.yml file!
